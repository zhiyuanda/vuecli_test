## 步骤如下：

### 一、安装

```
node -v
vue -V
```



![img](https://img-blog.csdnimg.cn/b980b7fb173f453eb8d2a7b3051cc2d0.png)

 ![img](https://img-blog.csdnimg.cn/a77f31b9abbf4bd09f1e428ce5827b81.png)
### 二、构建一个项目

语法：

```
vue init <template-name> <project-name>
```


 <template-name>：模板名称

<project-name>：项目名称

使用官方模板： 

```
vue init webpack vuecli_test
```
